# News-Article-Prediction-App
📰 News Article Prediction App

A Machine Learning-based web application that predicts the category/authenticity of news articles using Natural Language Processing (NLP) techniques. The app analyzes the input news text, processes it using text preprocessing methods, and applies a trained ML model to generate predictions in real time.

🚀 Features
Predicts news article categories or authenticity
Real-time text analysis
NLP preprocessing and feature extraction
Machine Learning model integration
User-friendly web interface
Fast and accurate predictions
🛠️ Technologies Used
Python
Scikit-learn
Pandas & NumPy
Natural Language Processing (NLP)
Streamlit / Flask
Joblib / Pickle
📊 Machine Learning Workflow
Data Collection
Text Preprocessing
Feature Extraction using TF-IDF
Model Training & Evaluation
Real-time Prediction through Web App
💡 Project Objective

The main objective of this project is to automate news article analysis using Machine Learning techniques and provide quick predictions based on textual content.
▶️ How to Run
pip install -r requirements.txt
streamlit run app.py
